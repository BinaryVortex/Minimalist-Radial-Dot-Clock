# ⏰ Minimalist Radial Dot Clock

Excited to share one of my latest front-end experiments --- a
**Minimalist Radial Dot Clock**.

This project reimagines how we visualize time by replacing traditional
clock hands and digital numbers with elegant radial dot indicators. The
goal was to create a clean, modern, and visually satisfying clock
experience that combines minimalism with interactive design.

## 💡 What is it?

The Minimalist Radial Dot Clock is a real-time clock visualization where
hours, minutes, and seconds are represented using animated dots arranged
in circular rings. Instead of focusing on conventional clock designs,
this project explores time through motion, geometry, and simplicity.

## 🚀 Why I built this

I wanted to challenge myself to create a time-based UI that feels
modern, elegant, and almost futuristic. Most clocks follow traditional
layouts, so this project was an opportunity to experiment with radial
interfaces, smooth animations, and minimal design principles while
keeping the experience lightweight and responsive.

## ✨ Features

-   🔵 Real-time radial dot visualization
-   ⏱️ Separate rings for hours, minutes, and seconds
-   🎨 Clean minimalist interface
-   📱 Fully responsive design
-   ⚡ Smooth animations and updates
-   🌙 Modern visual aesthetic
-   🖥️ Lightweight and performant implementation
-   🎯 Distraction-free time display
-   🔄 Dynamic DOM updates for efficiency
-   ✨ Elegant circular layout

## 🛠️ Tech Stack

-   🌐 HTML5
-   🎨 CSS3
-   ⚙️ JavaScript (ES6)
-   📐 CSS Variables
-   🎭 CSS Transforms & Animations
-   ⚡ requestAnimationFrame
